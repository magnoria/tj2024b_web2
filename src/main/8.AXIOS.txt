[ AXIOS ] : 비동기 통신
    - 대표적인 axios 함수 : Fetch(JS내장함수) vs axios vs ajax(Jquery)
    - 특징 : *JSON 타입으로 자동 변환*

    1. 설치 (각 환경이 다르다)
        JS(CDN) : <script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
        react(npm) : 방법이 다름

    2. 문법
        axios.HTTP메소드명('통신할HTTP주소' , {body})
            1. HTTP메소드명 : axios.get(), axios.post(), axios.put(), axios.delete()
            2. 통신할HTTP주소 : 스프링 환경에서는 RestController 에서 정의한 매핑 주소
                @GetMapping("/day04/student") ---> '/day04/student'
            3.body : 보낼 자료/데이터 객체{}
                    * post , put : body사용 / get, delete : body 사용하지 않는다.
            4. option : HTTP header 설정
                    * 기본값은 content-type: application/json


    3. 관례적인 유형들 : 여러개 통신을 순차처리 할 경우에는 주로 동기화 , 여러개 통신을 동시처리 할 경우에는 주로 비동기화

        1.비동기 유형1
            () => {
                axios.post( '', {} , {})
                    .then(response => console.log(response.data))
                    .catch(error => console.error(error));
            }


        2. 동기유형2 , .then / ,catch
            async () => {
             await axios.post( '', {} , {})
                 .then(response => console.log(response.data))
                 .catch(error => console.error(error));
        }

        3. 동기유형3 , .then X / .catch X
            async () => {
                try{
                     const response = await axios.post( '', {} , {})
                     console.log(response.data);
                     }catch(e) {console.error(e);}

                }
